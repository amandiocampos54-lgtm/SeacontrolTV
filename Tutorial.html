📺 Instruções de Instalação

MyTvOnline 2

1️⃣ Aceda a Informações do Sistema no menu principal.
2️⃣ Navegue até Data de Implementação.
⚠️ NÃO pressione OK! Apenas posicione o cursor.
3️⃣ No comando, pressione:
🔴 Vermelho → 🟢 Verde → 🟡 Amarelo → 🔵 Azul
4️⃣ Irá abrir o campo InputCode.
5️⃣ Insira o código fornecido.
6️⃣ Selecione Aplicar.
7️⃣ O dispositivo irá reiniciar automaticamente.

MyTvOnline 3

1️⃣ Abra Definições (ícone ⚙️).
2️⃣ Vá a Informações do Sistema.
3️⃣ Navegue até Data de Implementação.
⚠️ NÃO pressione OK! Apenas posicione o cursor.
4️⃣ No comando, pressione:
🔴 Vermelho → 🟢 Verde → 🟡 Amarelo → 🔵 Azul
5️⃣ Irá abrir o menu InputCode.
6️⃣ Insira o código fornecido.
7️⃣ Selecione Aplicar.
8️⃣ O dispositivo irá reiniciar automaticamente.
